# Cube Plans

Cube Cloud is available in four tiers. Cube Core (the open-source engine) is
free to self-host with no restrictions.

## Free

**Price:** Free forever

For hobbyists completing small projects and showcasing work.

- Connect any data source
- Model in semantic layer IDE
- Explore data in workbooks
- Publish workbooks as dashboards
- Semantic model agent
- Analytics chat
- Up to 5 workbooks
- 1,000 daily request limit

## Starter

**Price:** $40 per Developer / month

For professionals streamlining data workflows.

- Everything in Free, plus:
- Extended agent limits
- Premium LLMs access
- Unlimited workbooks
- Extended query limit
- Production deployment compute
- Cube Store caching
- Observability and high-availability

## Premium

**Price:** $80 per Developer / month

For teams scaling analytics and collaboration.

- Everything in Starter, plus:
- Explorer and Viewer roles
- Embedded dashboards
- Embedded analytics chat
- Unlimited queries
- 99.950% Uptime SLA
- Custom domains
- Multi-cluster deployment
- Performance Insights

## Enterprise

**Price:** Custom (contact sales)

For enterprises demanding security and support.

- Everything in Premium, plus:
- 99.990% Uptime SLA
- Dedicated single-tenant installation
- Bring Your Own Cloud (BYOC)
- Bring Your Own LLM (BYOLLM)
- SSO with SAML 2.0
- Workspace Access Control
- DAX API for Power BI
